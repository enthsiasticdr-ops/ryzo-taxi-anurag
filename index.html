Quick Help FAQ

How to verify a payment?

Scan the merchant QR code, enter the payment amount and the 12-digit UTR transaction ID in the Load Wallet form. Admin will verify and approve your request shortly.

How to resolve an OTP issue?

The OTP is generated securely on the server upon booking. The rider must enter this exact 4-digit code to start or complete the trip. If it fails, check that both devices are online.

What happens if I cancel?

Cancelling a ride after acceptance assigns a strike to the customer. Reaching 3 strikes results in temporary account suspension to keep the platform fair.
